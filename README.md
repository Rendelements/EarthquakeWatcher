# EarthqakeWatcher

Demo iOS project using the USGS earthquake API endpoint: Â https://earthquake.usgs.gov/earthquakes/feed/v1.0/

## Getting Started

Build and install on your favourite iOS device

### Prerequisites

```
Xcode 10.1
iOS 12.1
```

### Installing

You will need to add a Google Maps API key to the file EarthquakeWatcherGoogleInfo.plist\n
After pulling GIT repo, open using Xcode, and hit build...
```
Command - B
```

## Running the tests

```
Command - U
```
